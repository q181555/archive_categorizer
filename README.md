# 压缩包密码分类工具 (Archive Password Categorizer)

## 功能

扫描指定文件夹中的所有压缩包文件，用密码文件中的密码依次尝试，**不解压文件**，只验证密码。按密码将压缩包分类复制到目标文件夹中。

## 支持的格式

| 格式 | 支持方式 | 是否需要额外工具 |
|------|---------|----------------|
| `.zip`, `.jar`, `.war`, `.ear` | Python 内置 `zipfile` 模块 | ❌ 无需安装 |
| `.7z` | 可选 `py7zr` 或 7-Zip CLI | 推荐安装 7-Zip |
| `.rar` | 7-Zip CLI | 推荐安装 7-Zip |
| `.tar`, `.gz`, `.tgz`, `.bz2`, `.xz` | 通常无密码，仅复制到"未匹配"文件夹 | ❌ 无需安装 |

## 快速开始

```bash
# 1. 启动图形界面
python main.py

# 2. 界面操作
#    - 选择"源文件夹"：包含压缩包的文件夹
#    - 选择"目标文件夹"：分类结果存放位置
#    - 选择"密码文件"：txt 文件，每行一个密码（可用"创建示例密码文件"按钮）
#    - 点击"开始分类"

# 3. （可选）安装 7z 支持
#    下载安装 https://7-zip.org/ 并加入系统 PATH
#    或执行：pip install py7zr
```

## 输出目录结构

```
目标文件夹/
├── 密码：123456/
│   ├── archive1.zip
│   └── subdir/archive2.zip
├── 密码：password/
│   └── archive3.zip
└── 未匹配密码/
    └── unknown.zip
```

## 密码文件格式

```txt
# 这是注释行，会被忽略
123456
password
admin
# 空行也会被忽略

888888
```
